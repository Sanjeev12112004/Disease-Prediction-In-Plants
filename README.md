🌱 Disease Prediction in Plants

An AI-powered system for detecting plant diseases from leaf images using deep learning techniques. This project aims to assist in early diagnosis and improve agricultural productivity through automated disease classification.

📌 Overview

This project leverages Convolutional Neural Networks (CNNs) to analyze plant leaf images and classify them into healthy or diseased categories. By identifying patterns and visual symptoms, the system provides accurate predictions that can support timely intervention in crop management.

🧠 Tech Stack
Python
TensorFlow
Keras
OpenCV
NumPy, Matplotlib

⚙️ Key Features
Image-based plant disease detection
CNN-based classification model
Efficient preprocessing using computer vision techniques
Scalable and adaptable for multiple crop types
Simple and user-friendly execution

🚀 Getting Started

2. Install Dependencies

3. Run the Application
